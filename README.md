# Ghosts

A modern web application built with:

- **React** & **TypeScript**
- **Vite** for fast development
- **Convex** for database and authentication
- **Deployed on Netlify**

## Live Demo

[https://joyful-longma-df489c.netlify.app](https://joyful-longma-df489c.netlify.app)

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
